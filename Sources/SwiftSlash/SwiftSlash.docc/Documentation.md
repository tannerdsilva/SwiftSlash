#  ``SwiftSlash``

High performance concurrent shell framework built from the ground-up with async/await.

## About SwiftSlash

This version: `3.4.0`

Oldest recommend version: `3.3.1`

#### Platform Support
  - Linux
    - Full support

  - MacOS
    - Full support

  - Swift
    - SwiftSlash requires a Swift toolchain version >= `5.5`
